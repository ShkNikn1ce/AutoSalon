Бизнес правила:
1) Нельзя создать заявку на авто со статусом SOLID
2) После создания заявки, автомобиль автоматически переводится в RESERVED
3) Когда заявка получает статус APPROVED, создается Sale
4) После создания Sale автомобиль получает статус SOLD
5) На один автомобиль может существовать только одна активная заявка (NEW, IN_PROGRESS)
6) SWAGGER
7) Liquid Base, интеграция данных
